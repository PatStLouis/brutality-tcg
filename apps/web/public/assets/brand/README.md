# Brand assets

Commissioned Brutality artwork carried over from the legacy pack-opener
project (web-optimized WebP versions of the originals):

- `cardback.webp` — commissioned card back (shown during card reveals)
- `logo.webp` — Brutality wordmark (home page)
- `pack.webp` — gold foil pack wrapper (sealed pack on the opening page)

Originals live in the `PatStLouis/brutality-pack-opener` repo under
`src/assets/`. The `OldNewspaperTypes.ttf` display font was intentionally NOT
carried over until its license is confirmed.
